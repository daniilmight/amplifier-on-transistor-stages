# Усилитель переменного тока на транзисторных каскадах

**Краткое описание**  
Репозиторий содержит проект по разработке усилителя переменного тока для измерительного устройства на биполярных транзисторных каскадах (BC548). Включены расчёты h-параметров транзистора, проект двухкаскадного усилителя, схемы, результаты моделирования и вспомогательные материалы.

---

## Техническое задание и исходные данные
Проект разработан в рамках курсовой работы по дисциплине «Электронные устройства средств измерений».

- Модель транзистора: **BC548**.  
- Амплитуда входного сигнала: **18 mV**.  
- Внутреннее сопротивление источника сигнала: **80 Ω**.  
- Требуемая амплитуда выходного сигнала: **3.0 V**.  
- Сопротивление нагрузки: **1.9 kΩ**.  
- Напряжение питания: **18 V**.  
- Нижняя граничная частота: **180 Hz**.  

---

## Краткая сводка расчётов и результатов
(Подробные таблицы и вычисления — в документе и Mathcad-файлах.)

- Найдены h-параметры транзистора BC548 при трёх температурах (23, 25, 27 °C).  
  - Пример: при 27 °C получены: `h11 ≈ 949.696 Ω`, `h12 ≈ 0.000154`, `h21 ≈ 198.965`, `h22 ≈ 0.0000307 См`.
- Построен двухкаскадный усилитель: коэффициенты усиления каскадов ≈ `6.592` и `24.779`, итоговый коэффициент ≈ `163.343` (целевой ≈ 166.667).
- Подобраны номиналы разделённого резистора эмиттера и цепей смещения; подобраны развязывающие конденсаторы для обеспечения нижней граничной частоты ≈ `180 Hz` (стандартный номинал конденсаторов — 510 nF выбран по ряду E24).

---

## Схемы и моделирование
1. Открыть файлы схем в OrCAD.  
2. Выполнить DC-SWEEP / NESTED SWEEP для определения h-параметров (описанный в работе метод).
3. Выполнить AC-SWEEP для получения АЧХ и определения нижней граничной частоты.
